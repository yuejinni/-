# 报关文件生成器 使用说明

## 目录结构
```
报关项目/
├── customs_server/          ← Python 本地服务
│   ├── server.py
│   ├── excel_gen.py
│   ├── start_server.bat     ← Windows 双击启动
│   ├── start_server_mac.sh  ← Mac 启动
│   └── build_windows.bat    ← 打包成 .exe（Windows）
├── customs_extension/       ← Chrome 扩展
│   ├── manifest.json
│   ├── content.js
│   ├── sidebar.css
│   └── icons/
└── 报关产品基础资料旧.xlsx    ← 商品基础数据（需维护）
```

---

## 一、首次安装（每台电脑一次）

### 步骤 1：启动本地服务

**Windows（未打包版）：**
1. 确保安装了 Python 3.9+（官网下载：python.org）
2. 双击 `customs_server/start_server.bat`
3. 看到"✅ 报关服务已启动"即可

**Windows（已打包 .exe 版）：**
1. 双击 `start_server.bat`（会自动运行 server.exe）

**Mac（开发/测试）：**
```bash
bash customs_server/start_server_mac.sh
```

> 💡 服务只需启动一次，保持后台运行即可

---

### 步骤 2：安装 Chrome 扩展（一次性操作，约2分钟）

1. 打开 Chrome，地址栏输入：`chrome://extensions`
2. 右上角开启"**开发者模式**"
3. 点击"**加载已解压的扩展程序**"
4. 选择 `customs_extension/` 文件夹
5. 看到"报关文件生成器"出现在列表中即完成

---

## 二、日常使用

1. 确保 `start_server.bat` 已运行（绿点 = 在线）
2. 在筋斗云打开**调拨单**页面
3. 页面右侧出现绿色按钮，点击展开面板
4. 填写：汇率、发票号、保存路径
5. 点击"**🔄 刷新商品**"读取当前调拨单
6. 确认商品列表和价格无误
7. （可选）在"目标总额 USD"填入你要的总金额，系统自动调整单价
8. 点击"**生成文件 →**"
9. 在保存路径下找到生成的 3 个文件：
   - `出口报关单_LK00022.xlsx`
   - `Proforma_Invoice_LK00022.xlsx`
   - `Packing_List_LK00022.xlsx`

---

## 三、打包成 .exe 分发（Windows）

在已安装 Python 的电脑上：
1. 双击 `customs_server/build_windows.bat`
2. 等待打包完成，在 `dist_windows/` 里得到 `server.exe`
3. 将以下文件打包发给其他电脑：
   - `server.exe`
   - `start_server.bat`
   - `customs_extension/` 文件夹（整个）
4. 其他电脑**不需要安装 Python**，直接双击 `start_server.bat` 即可

---

## 四、维护基础资料

当有新商品时，在 `报关产品基础资料旧.xlsx` 中新增一行：

| 列 | 字段 | 说明 |
|----|------|------|
| B | 商品编号 | 筋斗云里的商品编码（必填）|
| E | 海关编码 | 10位HS编码 |
| F | 报关商品名称 | 中文报关用名 |
| G | 报关产品单位 | PAC / PCS / DZ |
| H | 境内货源地 | 如：义乌 |
| L | 毛重(KGS/包) | 每包毛重 |
| M | 净重(KGS/包) | 可用公式 =L2*0.9 |
| N | 尺寸(L×W×H cm) | 如：25*35*4 |

## 五、常见问题

**Q: 绿点是红色（服务离线）**
A: 双击 `start_server.bat` 重新启动服务

**Q: 商品显示"未找到"**
A: 该商品编码未在 `报关产品基础资料旧.xlsx` 中，需要手动添加

**Q: 价格显示 $0 或 "⚠ 备注无价格"**
A: 调拨单的备注栏需要包含 `¥XX` 格式，如 `8*20*12*¥89`

**Q: 想修改固定信息（买家名称/地址/港口）**
A: 编辑 `customs_server/excel_gen.py` 顶部的固定信息常量
